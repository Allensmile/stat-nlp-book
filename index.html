stat-nlp-book


About

stat-nlp-book project is an interactive Statistical Natural Language Processing book created for the purpose of the COMPGI19 course at University College London.

You can find the source code of the book here, together with instructions how to set it up interactively.

The project is actively pursued from 2016, and supersedes the stat-nlp-book-scala project.
